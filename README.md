# Glimpse Homebrew tap

```sh
brew install --cask glimpse-hq/tap/glimpse
```

Installs the Glimpse app and puts the `glimpse` CLI on `PATH`.

The cask in `Casks/glimpse.rb` is kept in sync with the latest release by a
workflow in [glimpse-hq/Glimpse](https://github.com/glimpse-hq/Glimpse); edit it
there, not here.
